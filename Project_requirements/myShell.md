### 基本需求：
- 实现 管道（也就是 | ）
- 实现 输入输出重定向( <  >   >>  )
- 实现 后台运行（ &） 
- 实现 内建命令（cd ） 比如：cd  ~ ，cd  - ，cd xxx
-  实现 tab补全 （提示:使用 readline 库）
### 进阶需求：
-  实现多管道命令，如：cat myshell.c |  grep ls  |  wc  -l 
- 屏蔽一些信号（如ctrl + c 不能终止）
- 通过设置环境变量，使程序可以直接使用 ( 也就是在任何地方都可以运行你的shell ) 
- 实现内置命令history
-  界面美观,输出整齐 
- shell 中一些带有颜色的命令将其颜色显示出来。比如：grep ,egrep,ls 等等
